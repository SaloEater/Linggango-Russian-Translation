- enigmaticlegacy\patchouli_books\the_acknowledgment

How to add translation for your language:
1) Download and install https://www.curseforge.com/minecraft/mc-mods/translations-extractor
2) Run the game
3) Open config/translations_extractor-client.toml
4) Change targetLanguage to your language code
5) Restart the game, load world, run `/extract translations lang`
6) Go to resourcepacks/ExtractedTranslations, copy assets and put them here into resourcepacks/<name of resource pack> (Currently they are divided by language)
7) Translate files
8) Submit to github https://github.com/Linggango/Linggango
